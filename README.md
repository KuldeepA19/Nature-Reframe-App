# 🌿 Nature Reframe

An instant mental-health companion for the modern age.

## ✨ Core Features
- **The Vent Box:** A private space to release negative emotions.
- **Positive Reframing:** Instant conversion of stressful thoughts into growth-oriented notes.
- **Nature UI:** A calming design aesthetic to reduce visual stress.

## 🎯 Target Audience
Designed specifically for young people looking for a quick, supportive way to handle daily emotional hurdles.

## 🛠️ Built With
- HTML5 / CSS3 (Nature Theme)
- JavaScript (Reframer Engine)
- Vercel (Cloud Hosting)
