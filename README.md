# 🌿 Nature Reframe

**Nature Reframe** is a wellness-focused Progressive Web App (PWA) designed to help users manage stress, anger, and loneliness. By "venting" thoughts into a digital garden, users can witness their emotional release transform into growth.

## 🚀 Live Demo
Check out the app live on Vercel: [https://nature-reframesoul-app.vercel.app/](https://nature-reframesoul-app.vercel.app/)

## ✨ Key Features
- **Digital Garden:** Every shared thought plants a new seed (emoji), creating a visual history of peace.
- **Mood Reframing:** Specialized UI themes based on emotional states (Anger, Stress, Loneliness).
- **PWA Ready:** Installable on mobile devices via `manifest.json`.
- **Zen Aesthetics:** A minimalist, nature-inspired design to lower heart rate and promote calm.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for accessibility.
- **CSS3:** Custom properties (variables) for dynamic mood-based themes.
- **JavaScript (Vanilla):** Light-weight logic for garden growth and local persistence.
- **Vercel:** Automated CI/CD deployment.
